# ResumeIQ Agent

This repository uses an assistant-style agent to manage project structure, production readiness, documentation, and test automation.

## Purpose
- Keep the ResumeIQ codebase modular and production ready.
- Ensure Python modules are cleanly separated by concern.
- Maintain frontend separation of templates and static assets.
- Keep documentation accurate and up to date.
- Support test automation and deployment helper files.

## Usage
- Run code and tests locally using `python -m pytest -q`.
- Use the Dockerfile to build container images.
- Use the environment example to configure production-ready values.

## Files
- `.agent.md` — agent instructions and repository guidance.
- `AGENTS.md` — list of available project agents.
- `Dockerfile` — container image build instructions.
- `docker-compose.yml` — local service orchestration helper.
- `.env.example` — recommended environment configuration.
